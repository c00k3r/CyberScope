# 🛡 CyberScope

Enterprise Security Posture Analyzer built using Java and JavaFX.

## Status

🚧 Planning & Research Phase

## Features

- Network Reconnaissance using Nmap
- System Hardening Assessment
- Vulnerability Mapping using CVE data
- Security Posture Scoring
- PDF Report Generation

## Technology Stack

- Java 17
- JavaFX 21
- AtlantaFX
- SQLite
- Apache PDFBox
- Jackson
- Maven
- Nmap
